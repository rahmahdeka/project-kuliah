Penelitian ini memodelkan dinamika jumlah pengguna aktif harian (Daily Active Users/DAU) pada platform e-learning Institut Teknologi Sumatera (ITERA) menggunakan pendekatan Rantai Markov. Data yang digunakan merupakan data log aktivitas harian selama semester ganjil tahun 2025 yang mencerminkan tingkat keterlibatan pengguna terhadap sistem e-learning. Data DAU didiskretisasi menjadi tiga state, yaitu rendah, sedang, dan padat, menggunakan metode IQR sebelum dianalisis lebih lanjut. Hasil penelitian menunjukkan bahwa matriks transisi Markov mampu menggambarkan probabilitas perpindahan antar state dengan baik, di mana setiap state memiliki kecenderungan tertentu untuk tetap atau berpindah ke state lain. Selain itu, perhitungan distribusi stasioner menunjukkan bahwa dalam jangka panjang sistem cenderung berada pada state sedang sebagai kondisi yang paling dominan. Hasil ini memberikan gambaran kuantitatif mengenai pola aktivitas pengguna e-learning serta dapat digunakan sebagai dasar dalam pengambilan keputusan berbasis data untuk pengelolaan sistem pembelajaran daring.
